#=> Instructions
The Jenkins integration assumes that there is an active working project.  To initialize an instance of the Jenkins class, send these parameters:
  # * +url+ - url for Jenkins server...
  # * +params+ - params hash
  # * +options+ - hash of options includes:...
  #   job_name - jenkins job name (from url)...
  #   username - jenkins user username...
  #   password - jenkins user password...
  #   verbose - true for verbose output (default = false)...
  #   simulate - simulate command - echo it (default = false)...
Like this:
jruby-1.6.7.2 :013 > @jenkins = Jenkins.new(jenkins_url, params, {"username" => "bbyrd", "password" => "private", "job_name" => "BRPM_4.6-D_WAR", "verbose" => true})
To change projects use the set_project
#> @jenkins.set_job_name("new_job")
